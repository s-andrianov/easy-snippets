# Easy Snippets

![Latest version](https://img.shields.io/badge/Version-0.1.0-green)
![Bootstrap version](https://img.shields.io/badge/Bootstrap-v5.3.8-blue)
![License](https://img.shields.io/github/license/s-andrianov/easy-snippets)

**Easy Snippets** — Bootstrap 5 code snippets extension for Visual Studio Code. Contains **272** snippets: atomic components, ready-made pages (login, register, dashboard, pricing, landing, e-commerce), and PHP/RedBeanPHP backend snippets.

> **Based on** [Bootstrap 5 Quick Snippets](https://github.com/anburocky3/bootstrap5-snippets) by Anbuselvan Rocky. Fully redesigned and republished under MIT license.

## Features

- **272 snippets** for Bootstrap 5.3.8
- **Full-page templates**: login, register, dashboard, landing, pricing, profile, contact, error pages
- **Large blocks**: hero sections, features, pricing tiers, testimonials, FAQ, team, timeline, gallery, footers
- **Admin components**: sidebar, topbar, data tables, kanban, inbox, charts
- **E-commerce**: product cards, cart, checkout, filters
- **PHP/RedBeanPHP**: auth system, CRUD, pagination, flash messages
- **Responsive by default**: all snippets use Bootstrap's responsive utilities

![Plugin in action](https://github.com/anburocky3/bootstrap5-snippets/blob/master/screenshots/help.gif?raw=true)

## Installation

1. Open VS Code
2. Press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (Mac)
3. Search for **Easy Snippets**
4. Click Install

## Usage

Type `bs5-` in HTML/PHP files and press `Ctrl+Space` to see all available snippets.

**Examples:**
- `bs5-button-default` → Bootstrap button
- `bs5-pages-login` → Full login page
- `bs5-admin-dashboard` → Complete dashboard layout
- `php-auth-login` → PHP login with RedBeanPHP (in `.php` files)

All snippets have tab stops — press `Tab` to jump between editable fields.
