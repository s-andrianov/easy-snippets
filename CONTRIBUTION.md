# How to Contribute

Contributions are welcome! This extension is a fork of [Bootstrap 5 Quick Snippets](https://github.com/anburocky3/bootstrap5-snippets) with significant enhancements.

## Contribution Guide

1. Fork this repository: https://github.com/s-andrianov/easy-snippets
2. Update templates in `src/templates/`:
   - Bootstrap components: `src/templates/bs5/`
   - PHP snippets: `src/templates/php/`
   - Icons: `src/templates/icons/<provider>/`
3. Run `bun run generate:snippets` to regenerate snippet files
4. Verify generated output:
   - `snippets/snippets.code-snippets`
   - `snippets/css.code-snippets`
   - `snippets/icons.code-snippets`
   - `snippets/php.code-snippets`
   - README snippets table
5. Test your snippets in VS Code
6. Push changes and create a Pull Request
